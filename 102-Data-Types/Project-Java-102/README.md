# نظام إدارة الطلاب

## الوصف
برنامج بلغة Java لإدارة قائمة الطلاب ودرجاتهم الأكاديمية. يسمح بإضافة طلاب جدد، عرض قائمة الطلاب مع درجاتهم، وحساب متوسط الدرجات. يستخدم البرنامج مفهوم الاستثناءات Try-Catch لمعالجة الحالات الخاصة مثل محاولة حساب المتوسط عند عدم وجود أي طلاب.

## المتطلبات
Java Development Kit JDK مثبت على الجهاز

## طريقة التشغيل
افتح Terminal داخل مجلد المشروع، انتقل لمجلد src بكتابة: cd src
بعدها اكتب: javac App.java
بعدها اكتب: java App
