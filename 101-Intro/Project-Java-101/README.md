# BMI Calculator (حاسبة مؤشر كتلة الجسم)

## الوصف
برنامج بلغة Java يحسب مؤشر كتلة الجسم (BMI) بناءً على الطول والوزن، 
ويصنف الحالة (نحافة / وزن طبيعي / زيادة وزن / سمنة).
كما يحسب ضريبة القيمة المضافة (VAT 15%) والمبلغ الإجمالي بناءً على قيمة BMI المدخلة.

## المتطلبات
- Java Development Kit (JDK) مثبت على الجهاز
